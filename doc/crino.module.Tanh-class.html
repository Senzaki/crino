Class Tanh

source code

A Tanh activation module computes its outputs with the non-linear element-wise hyperbolic tangent function, that can be defined as tanh(x) = [(exp(xi) − exp( − xi))/(exp(xi) + exp( − xi))]ni = 1, with x = [x1, x2, …, xn] ∈ ℝn.
